# Cryptofolio Ultra 🚀

A premium, feature-rich crypto portfolio tracker built with React.

## Features

- 🔐 **Authentication** — Login / Register system with localStorage
- 📈 **Real & Simulation Modes** — Practice trading with virtual ₹1,00,000
- 🕯️ **Pro Candlestick Charts** — Per-coin 7-day candlestick visualization
- 🏆 **Top Gainers/Losers** — Live market movers leaderboard
- 🔔 **Price Alerts** — Set above/below price triggers with notifications
- 🎮 **Trading Game Mode** — Paper trading with persistent leaderboard
- 📊 **Advanced Analytics** — Allocation breakdowns, top performers
- 🌐 **Global Market Overview** — Full market grid with sparklines
- 💡 **Smart Suggestions** — AI-powered portfolio recommendations
- 📅 **History Timeline** — Visual journey of all your trades
- ⏪ **Portfolio Replay** — Animated playback of your trading history
- 🎯 **Goal Tracker** — Set profit milestones with progress bars
- 🤖 **AI Chatbot** — Powered by Claude AI for portfolio advice
- 🎊 **Confetti Milestones** — Celebration on profit milestones
- 📤 **Export** — CSV and JSON portfolio export
- 💱 **Multi-Currency** — INR / USD / EUR / GBP support
- 🌙 **Premium Dark UI** — Syne + Space Mono + DM Sans typography

## Getting Started
